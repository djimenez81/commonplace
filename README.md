# Commonplace

**WARNING:** This project is at an extremely early stage, and I am doing it for
learning purposes. Therefore, at this point you are likely to find little to
nothing here of any interest, but feel free to look.

Commonplace is a _Note Taking Application_ project, started mostly as a learning
tool, that looks to implement an extensible platform to store your notes,
thoughts, projects, tasks and anything you can imagine.

## Features

At this point the road to travel is a bit fuzzy. Nevertheless, you look at the
following files:

- [Crazy ideas I have for this project](docs/IDEAS.md): As it says, it is a
bunch of things I have thought about that could be implemented, but have not
really been thought out deeply.
- [My motivation to start this project](docs/HISTORY.md): This is pretty much
personal stuff that motivated me to do this, in case it is interesting for
anyone.
- [Philosophy of implementation](docs/PHILOSOPHY.md): This document defines the
principles to guide the way the project is implemented.

## Technologies used

The project pretends to be implemented using the **Python** programming language
and it will make use of **YAML**, **MarkDown** and other languages.
